# 📱 Instagram Reels Clone

A front-end Instagram Reels UI Clone built using HTML, CSS, and JavaScript.

This project replicates the core UI and interactive behavior of Instagram Reels including like, follow, mute, and scroll-based reel transitions.

---

## 🚀 Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- DOM Manipulation  
- Event Handling  

---

## ✨ Features

- ❤️ Like / Unlike functionality  
- 👤 Follow / Unfollow toggle  
- 🔇 Mute / Unmute video  
- 📜 Smooth vertical scroll snapping  
- 🔁 Dynamic rendering using JavaScript  

---

## 📌 Video Files Notice

This project was originally built using locally stored video files inside the `videos` folder.

Due to GitHub’s 100MB file size limitation, the original video files could not be uploaded to this repository.

To run the project exactly as intended, the video files must be placed inside a local `videos` folder.

The UI, structure, and JavaScript logic remain fully functional.
